# Simple Banking System: Transaction Logic & History Tracking

[Italiano](#italiano) | [English](#english)

---

## <a name="italiano"></a> 🇮🇹 Versione Italiana
Applicazione bancaria che simula le operazioni di deposito e prelievo con validazione del saldo. Include una gestione dello storico transazioni per monitorare i movimenti del conto.

### 🛠️ Focus Tecnico
* **Validation:** Controllo preventivo per impedire saldi negativi.
* **Audit Trail:** Registrazione cronologica di ogni transazione.
* **Encapsulation:** Protezione dei dati finanziari sensibili.

---

## <a name="english"></a> 🇬🇧 English Version
A banking application simulating deposit and withdrawal operations with balance validation. Features transaction history tracking to monitor account movements.

### 🛠️ Technical Focus
* **Validation:** Pre-check logic to prevent negative balances.
* **Audit Trail:** Chronological logging of every financial transaction.
* **Encapsulation:** Protection of sensitive financial data.
